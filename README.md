# 百度网盘文件管理器

一个基于 PySide6 的百度网盘桌面客户端，提供**多标签页、分屏浏览、拖拽移动**等原生文件管理体验，弥补百度网盘官方客户端在文件管理方面的不足。

## 对比百度网盘官方的优势

| 功能 | 百度网盘官方 | 本系统 |
|------|-------------|--------|
| 多窗口浏览 | 不支持 | 支持多标签页 + 左右分屏 |
| 拖拽移动/复制 | 不支持 | 选中文件直接拖到目标目录 |
| 批量文件操作 | 操作繁琐 | 多选后一键移动/复制/删除/重命名 |
| 文件浏览 | 图标视图为主 | 详细列表视图，显示大小/修改时间 |
| 目录树导航 | 无 | 左侧目录树，快速跳转 |
| 右键菜单 | 功能有限 | 移动/复制/删除/重命名一应俱全 |
| 性能 | Web 技术栈，内存占用高 | 原生 Qt 控件，~60MB 内存 |
| 体积 | ~200MB+ | ~50MB |

## 功能特性

- **多标签页浏览** — 像浏览器一样开多个标签页，同时浏览不同目录
- **左右分屏** — 一键分屏，两侧独立操作，拖拽互传文件
- **拖拽移动/复制** — 选中文件拖到左侧目录树或另一侧面板，弹出菜单选择移动或复制
- **目录树导航** — 左侧完整目录树，懒加载子目录，点击即跳转
- **批量操作** — Ctrl/Shift 多选文件，工具栏一键移动/复制/删除
- **文件搜索** — 全局搜索，支持按类型筛选（文档/图片/视频/音乐/压缩包）
- **文件详情** — 底部详情面板显示文件名、路径、大小、MD5、修改时间
- **导航历史** — 支持前进/后退/向上一级
- **滚动加载** — 大目录自动分页加载，滚动到底部自动加载更多
- **Apple 风格 UI** — 采用 Apple Design Language，简洁清爽

## 使用方法

### 方式一：直接使用打包好的 EXE（推荐）

1. 下载仓库根目录下的 `百度网盘文件管理器.exe`
2. 双击运行即可，无需安装任何依赖

### 方式二：从源码运行

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行
python main.py
```

### 首次使用

1. 打开应用后，输入百度网盘的 **AppKey** 和 **SecretKey**（在 [百度网盘开放平台](https://pan.baidu.com/union/) 申请）
2. 点击"授权"完成登录
3. 即可浏览和管理网盘文件

## 操作说明

| 操作 | 方式 |
|------|------|
| 打开目录 | 双击文件夹，或点击左侧目录树 |
| 多选文件 | Ctrl+点击 / Shift+点击 |
| 移动文件 | 选中 → 工具栏"移动到..." 或拖拽到目录树 |
| 复制文件 | 选中 → 工具栏"复制到..." 或拖拽时选"复制" |
| 删除文件 | 选中 → 工具栏"删除" |
| 重命名 | 右键文件 → "重命名" |
| 搜索 | 顶部搜索框输入关键词 |
| 新建标签页 | 点击标签栏右侧 "+" |
| 分屏浏览 | 点击标签栏左侧分屏按钮 |
| 退出分屏 | 关闭右侧标签页或再次点击分屏按钮 |

## 打包为 EXE

```bash
pyinstaller --onefile --windowed --name "百度网盘文件管理器" --icon assets/icon.ico --add-data "assets;assets" main.py
```

## 技术栈

- **Python 3.10+**
- **PySide6** (Qt for Python) — 原生桌面 GUI
- **requests** — HTTP 请求百度网盘 API
- **PyInstaller** — 打包为独立 EXE
